🎁 Wishers — списки желаний и совместные подарки

Wishers — это кроссплатформенное приложение для создания списков желаний, бронирования подарков и организации совместных подарков с друзьями, семьёй и коллегами

Проект решает три основные задачи:

- где хранить идеи подарков
- как избегать повторяющихся подарков
- как удобно скидываться на дорогие подарки

📋 Списки желаний

- Создание личных списков для различных событий (день рождения, Новый год, свадьба и т.д.)
- Название списка, тип события и дата события.
- Просмотр, редактирование и удаление списков

🎁 Подарки

Ручное добавление подарков:

- Название, описание, примерная стоимость, категория, приоритет
- Редактирование и удаление подарков.
- Ограничение: не более 50 подарков в одном списке

🔒 Бронирование подарков

- Участники могут бронировать подарки, чтобы избежать дублей
- Просмотр статуса подарка: свободен / забронирован

💰 Виртуальная копилка

- Виртуальные взносы на общий подарок (без реальных платежей)
- Отображение текущей собранной суммы
